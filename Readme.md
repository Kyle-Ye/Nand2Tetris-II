# Nand2Tetris II
- A set of Kotlin implementation of Hack platform By Kyle
    - Jack Compiler
        - .jack -> .vm
    - Jack Analyzer
        - .jack -> .xml
    - VM Translator
        - .vm -> .asm
    - Hack Assembler
        - .asm(s) -> .hack
### Usage

`./bin/$(execuatable name) xx.file/folder`
